# Raster font renderer

Цей проєкт містить набір функцій для відтворення шрифтів у форматі (.c .h) видобутих з PSF шрифтів, із підтримкою масштабування, кирилиці, UTF-8 та малювання тексту із фоновими прямокутниками і контрастною рамкою.

---

## Особливості

- Підтримка Unicode (UTF-8), включно з кирилицею.
- Малювання окремих символів та тексту із масштабуванням.
- Автоматичний вибір кольору фону як інверсія текстового кольору з корекцією яскравості.
- Малювання фонового прямокутника з рамкою для тексту.
- Легко інтегрується з будь-якою системою малювання пікселів через callback `DrawPixel`.

---

## Файлова структура

- `glyphs.h` — оголошення функцій малювання, робота з UTF-8, структура шрифту.
- `glyphs.c` — реалізація функцій, включаючи пошук гліфа, масштабоване малювання, малювання тексту з фоном.
- `color_utils.h` і `color_utils.c` — допоміжні функції для роботи з кольорами: конвертація, інверсія, насиченість, вибір контрастного кольору.
- `font.h` — опис структури `Font`.
- Ваш основний код має викликати `Font_DrawTextScaled` або `DrawTextWithAutoInvertedBackground` для відображення тексту.

---

## Використання

### Ініціалізація

Оголосіть і ініціалізуйте шрифт типу `Font` зі своїм набором гліфів.

### Малювання тексту

```
Font_DrawTextScaled(&myFont, x, y, "Привіт світ!", spacing, scale, textColor, DrawPixel);
```

### Малювання тексту з фоном

```
DrawTextWithAutoInvertedBackground(&myFont, x, y, "Текст з фоном", spacing, scale, textColor, padding, borderThickness);
```

---

## Налаштування

- Параметри `spacing`, `scale`, `padding`, `borderThickness` можна налаштовувати для візуального вигляду.
- Колір тексту і фону базується на інверсії та контрасті, автоматично підбирається через `color_utils`.
